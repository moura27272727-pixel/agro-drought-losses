# Экономические убытки сельского хозяйства от засух

Учебно-исследовательский проект (НИУ ВШЭ): оценка влияния засух на урожайность
зерновых и **денежная оценка убытков** на примере России и сопредельных стран
черноморского / постсоветского зернового пояса.

> Все результаты получены на **реальных открытых данных** (FAOSTAT, World Bank CCKP,
> NOAA)
> [`data/README.md`](data/README.md).

## Постановка задачи
Связать межгодовую изменчивость урожайности зерновых с индикаторами засушливости
(SPI/SPEI, температура и осадки вегетационного периода, ENSO) и оценить
экономические потери от недобора урожая. Базовый уровень: корреляции; далее
панельные регрессии с фиксированными эффектами и модели машинного обучения.

## Данные
- **FAOSTAT**: урожайность, площади, валовой сбор, цены производителей (1992–2024).
- **World Bank CCKP**: месячные осадки и температура (CRU TS 4.07, 1901–2022) →
  расчёт индексов засухи **SPI** и **SPEI** (методология McKee 1993;
  Vicente-Serrano 2010).
- **NOAA ONI**: индекс Эль-Ниньо / Южного колебания (ENSO).
- Страны: **Россия (главный кейс)**, Украина, Казахстан, Беларусь, Молдова, Румыния.

## Методы
1. **Корреляции** (Пирсон/Спирмен) аномалий урожая с индексами засухи, по странам и
   в панели с фиксированными эффектами.
2. **Панельные регрессии** (`PanelOLS`, FE по странам и «страна×культура»,
   стандартные ошибки Дрисколла–Краая).
3. **Машинное обучение** (CatBoost, Random Forest, OLS) с кросс-валидацией
   (стандартной и строгой по годам); важность признаков и частные зависимости.
4. **Денежная оценка убытков**: недобор относительно технологического тренда (LOESS)
   × площадь × цена производителя.

## Ключевые результаты
- В панели с фиксированными эффектами связь засушливости и аномалии урожая значима:
  SPEI-6 r ≈ **0.42** (p < 0.001); +1σ SPEI ≈ **+4–6 %** к урожаю пшеницы, экстремальная
  жара лета ≈ **−7 %**.
- ML (CatBoost/RF) при стандартной кросс-валидации объясняет ~**37 %** дисперсии аномалий
  (OLS даёт 19 %); на целиком новых годах около 7 %. Климат значим, но не доминирует.
- Прямые потери сбора зерновых в России: **2012 г. ≈ 2.9 млрд $ (−18 %)**,
  **2010 г. ≈ 1.2 млрд $ (−15 %)**, **2021 г. ≈ 2.0 млрд $**; накопленно за
  2000–2022 гг. ≈ **6.5–11 млрд $** (диапазон по контрфактам).
- **Снятие главного ограничения:** климат зернового пояса (вместо усреднения по всей РФ)
  усиливает связь: корреляция с летней жарой растёт с 0.26 до **0.48** (p = 0.01).

## Структура репозитория
```
src/
  spei.py              расчёт PET, SPI, SPEI
  data_io.py           загрузка FAOSTAT / CCKP / ONI, конфигурация стран
  00_download_data.py  воспроизводимая загрузка всех данных
  01_build_panel.py    сборка аналитической панели
  02_correlations.py   корреляции + разведочные графики
  03_panel_regression.py  панельные регрессии FE
  04_ml_model.py       ML (CatBoost/RF/OLS), кросс-валидация
  05_economic_loss.py  денежная оценка убытков
  06_regional.py       климат зернового пояса (gridded), снятие ограничения
  run_all.py           полный прогон пайплайна
data/        исходные и обработанные данные (см. data/README.md)
figures/     графики (PDF для отчёта, PNG для слайдов)
report/      отчёт в LaTeX по ГОСТ + references.bib
slides/      презентация (beamer) + сценарий видео
```

## Запуск
```bash
python -m pip install -r requirements.txt
python src/run_all.py            # загрузка данных + весь анализ
# или, если данные уже скачаны:
python src/run_all.py --no-download
```
